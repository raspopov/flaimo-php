SMS/Mail-Ticker ChangeLog

Version History

Current Version: 1.003 (2003-06-03)

  • Added some get methods.

1.002 (2003-05-30)

  • Fixed body pasting bug.
  • From this release on the optional starting string can be at the beginning of the body message or the subject of the mail.
  • Added a method to add a ticker message manually (for example through a form - see sample script). Doesn't work in realtime though, since it takes some time until the message arrives in the mailbox.
  • Added a cooler javascript ticker to the sample page.

1.001 (2003-05-27)

  • Fixed connection bug.
  • Folder to connect to can now be specified in the ticker_settings.ini file.
  • It is now possible to allow only authors to post, which appear in a whitelist that can be defined in the ticker_settings.ini file.
  • Added option to backup old messages to a MySQL database.

1.000 (2003-05-24)

  • First Release

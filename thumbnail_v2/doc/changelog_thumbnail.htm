Thumbnail

What is it?

PHP classes for generating a thumbnail of a given picture, add a logo/watermark to it and cache it.

Version History

2.000 (2008-04-06)

  • Recursive function for checking folders now ignores folders that start with a dot.

1.003 (2004-03-13)

  • Corrected caching bug when requesting thumbs of a pic, but in different sizes.

1.002 (2003-10-30)

  • Added function getImagepaths() to the ContactSheet class which returns an array with all the paths to the cached thumbnails.

1.001 (2003-09-25)

  • Better error handling if picture file wasn't found.
  • It is possible to add multible logos/watermarks now.
  • Added a new class "ContactSheet", to create an picture index of a folder or manually added pics.
